FAQ

Questions people usually ask before they trust a PDF tool.

The answers below focus on privacy, browser behavior, and a couple of practical limits that matter in day-to-day use.

Are my PDFs uploaded anywhere?

The merge step runs in your browser, so the PDF contents are not sent to a server by the tool itself. The separate privacy page explains the remaining third-party font requests and the local PDF library bundle.

What if I accidentally add files in the wrong order?

Drag the queue rows into the right sequence before merging. The final PDF follows the order shown in the list.

Does this work with password-protected or damaged PDFs?

Not always. If a file cannot be read by the browser-side PDF library, the tool will stop and show an error for that file.
